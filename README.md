# Marine Insurance and Risk Management System

A comprehensive blockchain-based marine insurance platform built with Clarity smart contracts on the Stacks blockchain.

## Overview

This system provides end-to-end marine insurance operations including vessel registration, risk assessment, claims processing, emergency response coordination, and regulatory compliance management.

## Architecture

The system consists of five interconnected smart contracts:

### 1. Vessel Registry (`vessel-registry.clar`)
- Vessel registration and ownership tracking
- Survey scheduling and condition assessments
- Maintenance history and documentation
- Vessel status management

### 2. Risk Assessment (`risk-assessment.clar`)
- Multi-factor risk scoring algorithm
- Premium calculation based on risk factors
- Historical data analysis
- Risk mitigation recommendations

### 3. Claims Processing (`claims-processing.clar`)
- Claims submission and validation
- Automated processing workflows
- Settlement coordination
- Dispute resolution mechanisms

### 4. Emergency Response (`emergency-response.clar`)
- Incident reporting and tracking
- Resource deployment coordination
- Salvage operation management
- Emergency contact systems

### 5. Compliance Management (`compliance-management.clar`)
- Regulatory requirement tracking
- Safety certification management
- Audit trail maintenance
- Compliance reporting

## Key Features

- **Transparency**: All operations recorded on blockchain
- **Automation**: Smart contract-driven processes
- **Security**: Role-based access control
- **Efficiency**: Reduced processing times and costs
- **Compliance**: Built-in regulatory adherence

## Getting Started

1. Install dependencies: `npm install`
2. Run tests: `npm test`
3. Deploy contracts using Clarinet

## Testing

Comprehensive test suite using Vitest covering all contract functions and edge cases.
